# Touchdeck documentation

This folder holds the MDX sources for the Touchdeck docs site.

## Preview locally

1. Install the Mintlify CLI:

```
npm i -g mint
```

2. Run the dev server from this folder:

```
mint dev
```

3. Open http://localhost:3000 to view changes. The server reloads on save.

## Structure

- Content pages: *.mdx
- Navigation and theme: docs.json
- Assets: images/, logo/, snippets/

## Deploying

Push to the default branch to deploy.
